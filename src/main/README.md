|-- AndroidManifest.xml				安卓注册文件，注册各个活动与权限
|-- README.md
|-- assets										  静态资源文件夹
|-- java											  代码文件
`-- res												各种布局文件与图片资源文件





assets中存放的是静态资源文件，运行过程中不需要修改的文件，比如四史、电子书等

res中存放的是安卓的各个页面的布局文件以及图片资源

java下是代码文件

AndroidManifest.xml是安卓的核心文件，在里面注册网络权限、sd卡读写权限等，以及为每个活动进行注册，并设置主活动（开机运行的活动）
